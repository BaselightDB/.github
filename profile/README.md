**Baselight** is the unified data platform for humans and AI. We host thousands of high-quality, structured datasets — spanning finance, demographics, climate, sports, healthcare, and more — all queryable with SQL in a single workspace.

Think of it as GitHub for structured data: explore public datasets, upload your own, write queries, build charts, and share insights — no infrastructure required.

### 🚀 Get started

- **[baselight.ai](https://baselight.ai/)** — Learn more about Baselight
- **[baselight.app](https://baselight.app)** — Sign up and start querying
- **[Catalog](https://baselight.app/catalog)** — Browse 70K+ tables from Our World in Data, World Bank, CDC, Eurostat, and much more
- **[Docs](https://baselight.ai/docs/getting-started/)** — Getting started guide, SQL reference, and API docs
- **[MCP Server](https://baselight.ai/docs/connecting-to-the-baselight-mcp-server/)** — Connect Baselight to Claude, Cursor, and other AI tools

### 🌈 Community

We'd love your help making Baselight better. Star your favorite datasets, fork and remix queries, and share insights publicly. If you spot an issue with a dataset or have a suggestion, reach out — we read everything.
